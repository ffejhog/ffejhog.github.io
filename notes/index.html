Merge sort

Merge(A,B,P,q,r)
//Precondition: A[p...q], A[q+1...r)] are sorted
//B is for temp work
Copy A[p...r] into B[p...r]
i = p
j = q+1
for k=p to r
    if j > r or (i <= q and B[i] <= B[j])
        A[k] = B[i++]
    else
        A[k] = B[j++]
