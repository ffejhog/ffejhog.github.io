Prim's Algorithm

Input: G=(V,E) weights w, Root s

Prim(G,w,s)
    for each u in V
        u.d = infinity
        u.pi = null
    s.d = 0
    Set = null
    wmst = 0
    Create a priority queue Q with all vertices (priority = u.d)
    while Q is not empty
        u = Q.remove()
        Set = Set ∪ {u}
        wmst = wmst + u.d
        for each v in adj[u]
            if v is in Q and w(u,v) < v.d
                v.d = w(u,v)
                v.pi = u
    return wmst
